# What is this thing?

This a is a cmake'ized template for writing application on the GD32F303CDET6. It contains the Standard Peripheral Library + USBD Lib (untested) and maybe FreeRTOS in the future.

# How to build?

```
# mkdir build
# cd build
# cmake ..
# make hello_world
```
